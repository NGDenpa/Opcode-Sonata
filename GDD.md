# Machine Loop - 游戏设计文档

## 核心概念
**主题**: Machine - 机械循环、自动化流水线
**类型**: 节奏解谜 + 编程游戏
**时长**: 1-2周开发 | 2-3人团队

## 玩法循环

### 阶段1: 规划编程
- 玩家编写炮台发射节奏（loop语言）
- 玩家编写水管旋转序列（loop语言）
- 可视化预览管道布局

### 阶段2: 执行观察
- 按下播放，系统按tick执行
- 可暂停、单步调试
- 可实时微调参数（高级功能）

### Tick执行顺序
```
1. 水管旋转（按loop指令）
2. 炮台发射判定（按loop指令）
3. 子弹移动（每个前进1格）
4. 碰撞检测（子弹vs水管墙壁）
5. 目标命中判定
```

## 胜利条件
- 子弹命中目标N次（关卡可配置，如10次、20次）
- 可选：限制总tick数（时间压力）
- 可选：评分系统（Perfect/Good/Miss节奏评级）

## Loop语言设计

### 炮台节奏语言（简谱风格）
```
# 示例：4拍循环
1---  # 第1拍发射，后3拍静默
--1-  # 第3拍发射
1-1-  # 第1、3拍发射
```

### 水管旋转语言（角度控制）
```
# 示例：旋转序列
R90   # 顺时针90度
L90   # 逆时针90度
---   # 保持不动
R90,---,L90,---  # 4拍循环
```

## 关卡元素

### 炮台
- 固定位置
- 可配置发射方向
- 执行loop节奏

### 水管类型
- 直管（I型）：上下通过
- 弯管（L型）：90度转向
- 三通（T型）：三个方向
- 十字（+型）：四个方向

### 子弹
- 每tick移动1格
- 碰到墙壁消失
- 通过水管内部继续前进

### 目标
- 固定位置
- 记录命中次数

## 美术风格
- 极简工业风
- 网格背景
- 霓虹色彩（炮台=红、子弹=黄、目标=绿）
- 水管用线框表示

## 音效/音乐
- 背景：循环电子节拍（与tick同步）
- 发射：合成器音效
- 命中：打击乐
- 旋转：机械音

## 技术栈
- **原型**: HTML5 Canvas + JavaScript
- **最终**: Godot 4.x
- **资产**: 矢量图形 + 程序化生成

### Week 2

## 风险点
- Loop语言学习曲线（需要教程关卡）
- 碰撞检测复杂度（水管旋转时）
- 节奏感不足（需要强音效反馈）
